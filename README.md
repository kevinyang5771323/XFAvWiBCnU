# 项目简介

## 核心功能
- 基于YOLOv8的汽车表面损伤实例分割检测系统
- 支持图片、视频及摄像头进行检测
- 提供UI界面，实时显示目标位置、目标总数、置信度等信息
- 支持批量检测并直接查看所有检测结果
- 支持检测结果保存
- 可自定义置信度和IOU阈值
- 可选择显示标签和原图
- 可选择不同检测模型
- 提供11000张数据集，包含['Car-Damage']一类数据

## 适用场景
- 汽车损伤检测
- 车辆维修评估
- 汽车质量检测
- 车辆保险评估
- 汽车生产质量控制

## 技术特点
- 使用Pycharm+Anaconda或VSCode+Anaconda开发环境
- 基于Python 3.9，依赖opencv-python、PyQt5、ultralytics、torch1.9等技术栈
- 已训练好的模型，配置环境后可直接使用
- 包含完整程序文件、UI界面源文件、图标和数据集
- 提供项目运行教程
